**终端 1 — 启动后端 (FastAPI)**

```powershell
.\.venv\Scripts\Activate.ps1
python backend/main.py
```

后端地址: http://127.0.0.1:8000 | API 文档: http://127.0.0.1:8000/docs

**终端 2 — 启动前端 (Streamlit)**

```powershell
.\.venv\Scripts\Activate.ps1
streamlit run frontend/app.py
```

前端地址: http://localhost:8501
